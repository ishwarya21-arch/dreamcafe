# ☁️ Dream Cafe Website 🍰

Dream Cafe is a responsive restaurant website created using HTML, CSS, and JavaScript with a light pink and white theme.


## How to Run

1. Download all files
2. Create a folder named:

```text
DreamCafe
```

3. Place these files inside:

- index.html
- style.css
- script.js
- README.md

4. Open:

```text
index.html
```

in your browser.

---

## Website Pages

### Home
Displays cafe banner and welcome section.

### Menu
Contains food items:

- Burger Combo 🍔
- Pizza 🍕
- Biriyani 🍛
- Healthy Salad 🥗
- Mutton Manthi
- South Indian Breakfast
- Pancake Icecream 🍨

### Booking
Allows users to book a table.

### About
Information about Dream Cafe.

### Contact
Allows users to send messages.
